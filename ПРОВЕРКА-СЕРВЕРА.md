# Проверка существующего кода на сервере

## Команды для проверки:

### 1. Посмотрите PHP файлы в www (SSH):

```bash
ls -la ~/www/*.php
```

### 2. Найдите файлы, которые работают с CRM:

```bash
grep -r "datapoint.center" ~/www/*.php
grep -r "auth_login" ~/www/*.php
```

### 3. Посмотрите структуру проекта:

```bash
ls -la ~/www/
cat ~/www/test_crm.php  # или другой файл, который работает с CRM
```

### 4. Если найдете рабочий файл логина, покажите его содержимое:

```bash
cat ~/www/название_файла.php
```

## Что ищем:

- Функции логина в CRM
- Рабочий код для получения cookies
- Функции для парсинга данных из CRM

Если найдете рабочий код, я адаптирую его для нашего API.
