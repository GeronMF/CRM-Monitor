## Быстрый старт (3 шага)

### 1. Соберите проект

```bash
npm install
# Создайте .env с Supabase данными
npm run build
```

### 2. Скопируйте файлы на сервер

**Windows PowerShell:**
```powershell
scp -P 32762 -r dist/* aiecom_uploadspace@decloud2376.zahid.host:/home/aiecom_uploadspace/public_html/crm-monitor/
```

**Или через Git Bash:**
```bash
scp -P 32762 -r dist/* aiecom_uploadspace@decloud2376.zahid.host:~/public_html/crm-monitor/
```

**Если директория другая, сначала проверьте:**
```bash
ssh -p 32762 aiecom_uploadspace@decloud2376.zahid.host "ls -la ~"
```

### 3. Настройте базу данных

**Через phpMyAdmin:**
1. Откройте `https://ai.ecom-upload.space/phpmyadmin`
2. Войдите: пользователь `aiecom_uploadspace`, пароль `73762984-d17c-45cd-b77c-fad42de47b65`
3. Выберите БД `aiecom_uploadspace`
4. SQL → выполните содержимое файла `setup-database.sql`

**Или через SSH:**
```bash
ssh -p 32762 aiecom_uploadspace@decloud2376.zahid.host
mysql -u aiecom_uploadspace -p aiecom_uploadspace
# Введите пароль: 73762984-d17c-45cd-b77c-fad42de47b65
# Затем выполните SQL из setup-database.sql
```

## Настройка Nginx

Если у вас есть доступ к конфигурации:

```bash
ssh -p 32762 aiecom_uploadspace@decloud2376.zahid.host
# Найдите конфиг (может потребоваться sudo)
sudo nano /etc/nginx/sites-available/ai.ecom-upload.space
```

Добавьте (замените путь если нужно):

```nginx
location /crm-monitor {
    alias /home/aiecom_uploadspace/public_html/crm-monitor;
    try_files $uri $uri/ /crm-monitor/index.html;
    
    location ~* \.(js|css|png|jpg|jpeg|gif|ico|svg|woff|woff2|ttf|eot)$ {
        expires 1y;
        add_header Cache-Control "public, immutable";
    }
    
    location ~* \.html$ {
        expires -1;
        add_header Cache-Control "no-store, no-cache, must-revalidate";
    }
}
```

Проверьте и перезагрузите:
```bash
sudo nginx -t
sudo systemctl reload nginx
```
